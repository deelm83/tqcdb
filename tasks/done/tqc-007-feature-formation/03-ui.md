## 7. Accessibility

- All interactive elements have visible focus states
- Color is not the only indicator (use icons + text)
- Minimum touch target size: 44x44px
- Keyboard navigation support for drag-drop (arrow keys to move)
- Screen reader announcements for conflict status changes
