# UI Specifications

## Crop Modal

### Layout
```
┌─────────────────────────────────┐
│ Cắt ảnh đại diện            [X] │  ← Header
├─────────────────────────────────┤
│                                 │
│    ┌─────────────────────┐      │
│    │                     │      │
│    │    Crop Area        │      │  ← 7:10 ratio
│    │    (draggable)      │      │
│    │                     │      │
│    └─────────────────────┘      │
│                                 │
│  Thu phóng [━━━━●━━━━━━━]       │  ← Zoom slider
│                                 │
├─────────────────────────────────┤
│              [Hủy]  [Cắt ảnh]   │  ← Footer buttons
└─────────────────────────────────┘
```

### Styling (Tailwind)
- Modal: `bg-stone-800 border-stone-600 rounded-lg max-w-md`
- Header: `text-stone-100 border-b border-stone-600`
- Crop area: `bg-stone-900` with amber border on selection
- Buttons: Cancel = `text-stone-300`, Confirm = `bg-amber-600`

### States
1. **Default**: Image loaded, ready to crop
2. **Dragging**: User adjusting crop area
3. **Processing**: "Đang xử lý..." with spinner
4. **Error**: Red text below crop area

## Re-crop Button

Add next to existing image preview on edit page:

```
┌──────────┐
│  [image] │  [Cắt lại]
└──────────┘
```

Button: `text-sm text-amber-500 hover:text-amber-400`
